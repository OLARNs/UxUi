01 — Foundations

Design Principles

Five principles that govern every design decision in REAVER. When in doubt, return to these.

01
Precision
Every element is intentional. No decorative noise. Each component earns its place through function alone.
02
Trust
Consistent patterns, clear hierarchy, reliable feedback. Users always know where they are and what will happen next.
03
Speed
Optimized for fast transactions. Minimal clicks, clear CTAs, zero cognitive friction between intent and action.
04
Premium
Elevated through restraint — generous spacing, sharp contrast, obsessive consistency. Never through decoration.
05
Clarity
Content-first layout. Game artwork provides context; it never competes with the interface. Actions are always the visual priority.
